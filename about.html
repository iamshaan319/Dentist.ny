ABOUT THIS PROJECT

How We Built
NY Dental Directory

🏛️

NPI Registry

The primary source. Every licensed dentist in New York State is registered in the US federal National Provider Identifier database maintained by CMS. This is free, public, and official government data.

🗺️

Google Places API

Secondary source enriching records with ratings, review counts, office hours, websites, and GPS coordinates from Google Maps — updated in real time.

🔄

Auto De-duplication

Records from both sources are matched by name, phone, and ZIP code. Where the same provider appears in both, the best available data from each source is merged into a single record.

⚡

GitHub Backend

All dentist data is stored as a JSON file in a public GitHub repository. The website fetches data directly from GitHub's raw content CDN — no server required, zero hosting cost.

How to Update the Data

# 1. Run the Python scraper

python Doctor.py

# 2. Convert the Excel output to JSON (using the converter script)

python excel_to_json.py ny_dentists.xlsx data/dentists.json

# 3. Push to GitHub — the website auto-updates

git add data/dentists.json && git commit -m "Update dentist data" && git push
